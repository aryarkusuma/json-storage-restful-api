### Endpoint
#### Storing JSON (POST)
> /api/save-json
#### Retriving stored JSON (GET)
> /api/get-json?id=:id
#### Delete stored JSON (GET)
> /api/del-json?id=:id

These are build on the top of netlify + upstash (redis hosting)
